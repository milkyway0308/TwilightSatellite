## 소개
TwilightSatellite는 [Velocity](https://papermc.io/software/velocity)와 [Spigot](https://www.spigotmc.org/)에 기반하는 안정적인 프록시 중계 프레임워크이자 올인원 유틸리티 플러그인입니다.

어노테이션으로 최소화된 자동화 절차를 다수 포함합니다.
